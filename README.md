# Media File Upload System

A cross-platform media file upload solution supporting both web and mobile platforms with advanced chunked upload capabilities, magic number validation, and MD5 deduplication.

This repository contains three main components, each with its own detailed documentation:

1. **[Web Frontend](./web/README.md)**: A Next.js/React application for web-based uploads.
2. **[Mobile App](./mobile/README.md)**: A React Native Expo application for iOS and Android uploads.
3. **[Backend API](./backend/README.md)**: A PHP/Symfony 6 server handling chunk reception, deduplication, and storage.

Please navigate to the respective directories above for specific installation, configuration, and testing instructions for each platform.
